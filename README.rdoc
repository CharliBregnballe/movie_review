This app is a movie review app.
It contains a movie model, a review model, a user model and a search functionality.
There is also a star rating plugin (raty) for the reviews.

Movies contain title, description and image (paperclip)
Reviews contain a 5-star-rating and a review.
For the search the gem searchkick is used along with elasticsearch.

Custom gems:
devise
paperclip
bootstrap-sass
searchkick
